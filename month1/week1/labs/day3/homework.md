# Homework Day 3

## Question 1
What is a string?
Answer: A string is an immutable sequence of characters

## ## Question 2
Why is a string called a sequence?
Answer: A string is called a sequence because its characters are stored in a specific order, and each character can be accessed by its index.

## Question 3
What does name[0] mean?
Answer: It retrieves the character stored at index 0 of the string object referenced by name.

## ## Question 4
Why are strings immutable?
Answer: Strings are immutable because you cannot modify the string once it has been created.

## Question 5
Explain slicing in your own words.
Answer: slicing is the process of extracting a portion of a sequence using start and end indices.

## Question 6
Why does python choose strings to be immutable?
Answer: Because of benefits like: 
- Memory efficiency.
- Safety (shared strings can't be accidentally modified).
- Hashability (important for dictionary keys)
