# Homework Day 1

## Question 1

What is programming?

Answer: Programming is problem solving before you write any code. Programming is basically giving the computer a set of instructions to solve a problem. Programming is the process of designing, writing, testing, debugging and maintaining instructions that enable a computer to solve a problem.

## Question 2

What is the difference between programming and coding?

Answer: Programming is the entire problem solving process including the thinking and writing the code. Coding is just one step. It is one part of programming. It involves writing the code that solves the problem.

## Question 3

Explain an algorithm in your own words.

Answer: An algorithm is a finite and ordered sequence of instructions/steps that is used to solve a problem.

## Question 4

Why do computers need precise instructions?

Answer: Computers need precise instructions because computers cannot infer missing information or make assumptions like humans can.

## Question 5

Why shouldn't programmers immediately start writing code?

Answer: Programmers shouldn't start writing codes because writing code is just one part of the problem solving process. They need time to think about the entire programming process end to end and figure out every step before writing codes to solve the problem
