# Homework Day 4

## Question 1
What is a list?
Answer: A list is a mutable ordered sequence of objects

## Question 2
What is the difference between a list and a string?
Answer: Lists are mutable collections of objects, while strings are immutable sequences of characters. Lists are designed for data that changes, whereas strings are designed for text.

## Question 3
Why are lists mutable?
Answer: Python made lists mutable because many real-world collections naturally change over time. For example, shopping carts, user lists, playlists, and queues need items to be added, removed, or updated efficiently.

## Question 4
When would you use a list instead of a string?
Answer: You use a list when you will make modifications to the elements. For instance: Shopping cart, To-do list, Machine learning dataset rows, Queue of jobs, List of transactions

## Question 5
What happens when you call append()?
Answer: append adds the element to the end of the list
